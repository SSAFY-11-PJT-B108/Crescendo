# 팬 참여형 아이돌 커뮤니티

## 시작하기

```
git clone [URL]
npm install
cd Crescendo
npm start
```

## 코딩 컨벤션

**태그: 제목의 형태**

| 태그 이름 |                                설명                                 |
| :-------: | :-----------------------------------------------------------------: |
|   feat    |                      새로운 기능을 추가할 경우                      |
|    fix    |                          버그를 고친 경우                           |
|   chore   |                 빌드 업무 수정, 패키지 매니저 수정                  |
|   docs    |                              문서 수정                              |
|   init    |                              초기 설정                              |
|   test    |                             테스트 코드                             |
|   style   | 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우, 짜잘한 코드 수정 |
| refactor  |                            코드 리팩토링                            |
|   merge   |                             코드 합치기                             |
