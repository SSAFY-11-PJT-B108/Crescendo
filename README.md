# 팬 참여형 아이돌 커뮤니티

## 시작하기

```
git clone [URL]
npm install
cd Crescendo
npm start
```

## 코딩 컨벤션

**[태그] 제목의 형태**

| 태그 이름 |                                설명                                 |
| :-------: | :-----------------------------------------------------------------: |
|   FEAT    |                      새로운 기능을 추가할 경우                      |
|    FIX    |                          버그를 고친 경우                           |
|   CHORE   |                 빌드 업무 수정, 패키지 매니저 수정                  |
|   DOCS    |                              문서 수정                              |
|   INIT    |                              초기 설정                              |
|   TEST    |                             테스트 코드                             |
|   STYLE   | 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우, 짜잘한 코드 수정 |
| REFACTOR  |                            코드 리팩토링                            |
